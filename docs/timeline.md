# Timeline


## Week 0 - Prepare:

* Learn OpenCV
* Learn Multithreading
* Improve Knowledge of Flask and Flutter conectivity
* Order Items

## Week 1 - Board Recognition:

* Get openCV working on the PI
* Work on getting the PI to recognise a chess board
* Get the PI to understand the grid system of the board

## Week 2 - Piece and Move Recognition:

* Train the system in learning what chess pieces look like
* Get the system to recognise the pieces from a top down view
* Then have it recognise a move based on 2 different video frames

## Week 3 - Implement the Event Bus and State Manger:

* Have different possibilities be handled by the Event Bus
* Have the state manager hold all states of the game so far
* Have the Event Bus be able to send new updates to the board state

## Week 4 - Implement the AI Engine and add Continutiy:

* Work on getting information from stockfish based on a given game state
* Also Have it rate a human played moves when in review mode
